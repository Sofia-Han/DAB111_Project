About

This dataset was provided by Chase Willden and can be found it the Data.World Website through the following link: https://data.world/chasewillden/web development-courses-from-udemy/workspace/file?filename=WebDevelopment.csv The dataset in original csv format displays 1200 records about Web Development Courses from the learning plataform Udemy. This website provides information about the courses names, payment form (if a course is free it shall display 'True' and if is paid, it shall display "False"), course's price, course's number of subscribers, number of reviews provided, number of published lectures and instructional level (begginer, intermediate, expert or all levels) in this particular order.
